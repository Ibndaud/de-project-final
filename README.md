# 🍽 Streaming Service for Time-Limited Restaurant Campaigns

## 📖 О проекте

В рамках проекта реализован стриминговый сервис обработки рекламных кампаний ресторанов с ограниченным сроком действия.

Сервис получает события о новых акциях из **Kafka**, определяет подписчиков ресторанов на основе данных из **PostgreSQL**, формирует персонализированные сообщения и:

- ✅ отправляет их в выходной Kafka-топик для сервиса push-уведомлений;
- ✅ сохраняет записи в PostgreSQL для последующего анализа.

Проект демонстрирует работу с потоковой обработкой данных в реальном времени с использованием **PySpark Structured Streaming**.

---

## 🏗 Архитектура

### Используемые технологии

- **PySpark Structured Streaming** — потоковая обработка данных  
- **Apache Kafka** — брокер сообщений (input/output топики)  
- **PostgreSQL** — хранение подписчиков и аналитическая таблица  
- **Docker / Docker Compose** — локальная инфраструктура  
- **Helm** — деплой сервиса Kafka - PostgreSQL
